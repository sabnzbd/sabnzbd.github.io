--- title: Install SABnzbd for Windows from source ---

Running the source python (.py) files on a window box is not recommended unless you want to try the latest GitHub copy, or make changes yourself.

  1. Install Python 2.7.x from Python.org
    Make sure to select during the installation Add python.exe to path!

  2. Install Git for Windows on your system. Defaults options are OK

  3. After installing both, restart your computer and open a Command Prompt.
    Run these commands: 

    pip install cheetah
pip install cryptography
pip install subprocessww
pip install pypiwin32
pip install sabyenc

  4. Get a local copy (clone) of the SABnzbd source by opening Git Bash from the start menu and running: 

    git clone https://github.com/sabnzbd/sabnzbd.git
    Press "Insert" instead of "Ctrl+V" to paste into Git Bash.

    You can specify the target directory as a second argument if you donâ€™t want it in the default location (C:\Users\USERNAME\sabnzbd\).

  5. To enable multi-language support type in the Git Bash you still have open: 

    cd sabnzbd
python tools/make_mo.py

  6. Open a Command Prompt. If you did not change the location of the Git clone, you can now start SABnzbd by running 

    python C:\Users\USERNAME\sabnzbd\SABnzbd.py

Optional to update the source files to the latest version, open Git Bash and type: 

cd sabnzbd
git pull

Optional (not show the DOS command line when SABnzbd is launched):

  • Create a .cmd file to easily launch SABnzbd such as: python "path\to\folder\SABnzbd.py"

  • Put a shortcut to the .cmd file in your startup folder, and set it to start as minimized.
